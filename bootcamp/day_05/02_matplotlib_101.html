intro to matplotlib¶

Matplotlib is the most common charting package, see its documentation for details, and its examples for inspiration.

Setting Styles¶

We will use the plt.style function to set the appropriate aesthetic styles for all of our figures. I am using the ggplot style, which ensures that the plots we create use the nice and calm design schemes

plt.style.use('ggplot')

Object-oriented interface¶

The object-oriented interface is for when you want more control over your figure. The object-oriented interface the plotting functions are methods of explicit Figure and Axes objects.

We can create helper functions to make our plots consistent in style¶

def change_style(ax,xlim=None, ylim=None, xlabel='X label', ylabel='Y label'):

    ax.set_xlabel(xlabel)
    ax.set_ylabel(ylabel)
    
    ax.grid(linestyle="--", linewidth=0.5, color='.25', zorder=-10)

    if type(ylim) is list:
        ax.set_ylim(*ylim)
    if type(xlim) is list:
        ax.set_xlim(*xlim)
    
    ax.xaxis.set_major_locator(LinearLocator(5))
    ax.xaxis.set_minor_locator(AutoMinorLocator(4))
    ax.yaxis.set_major_locator(LinearLocator(5))
    ax.yaxis.set_minor_locator(AutoMinorLocator(4))
    ax.spines[['bottom', 'left']].set_visible(True)
    ax.spines[['bottom', 'left']].set_color('k')
    
    ax.tick_params(which='major', width=1.0)
    ax.tick_params(which='major', length=10)
    ax.tick_params(which='minor', width=1.0, labelsize=10)
    ax.tick_params(which='minor', length=5, labelsize=10, labelcolor='0.25')

Often we would like to add letters per panel¶

def add_panel_letter(fig, ax, letter ,pos=( 0, 1.05), fontsize=12):
    ax.text(*pos, 
            letter, 
            ha='left',
            fontsize=fontsize,
            fontweight='bold', 
            transform=ax.transAxes)

Finally this framework is limited by your imagination¶
